# GamingHub

A senhora deverá instalar em seu PC o node.js, após isso será necessário a execução do código a seguir no prompt de comando: "npx create-react-app gaminghubsite --template redux". Após a instalação, abra os arquivos no Visual Studio Code e no terminal execute o seguinte comando: "npm install @material-ui/core", após a instalação execute "npm start" e o site abrirá em seu navegador.
